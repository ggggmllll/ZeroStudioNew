###tree-sitter-ndk detailed documentation

##Introduction: 
The language projects within tree-sitter-ndk are completely developed in a way that is independent of the original author's approach. They use complex methods such as Rust and npm to generate and build tree-sitter-xxx.
The tree-sitter-ndk build process is very simple. It only requires Gradle + NDK + CMake to quickly build the tree-sitter for the desired language.

##Instructions for use
#+1. Select the desired TS language, and extract all the C/C++ programming language files in the "src/" directory to the "/src/main/cpp" folder of the Gradle module.
The core must extract the file.：src/tree_sitter/*.h，*.c and parser.c, scanner.c

#+2. Use the standard template and import it into src/main/cpp. Take toml as an example here.
PS: The content in the template, configuration files, etc. should be modified as appropriate.

src/main/cpp/CMakeLists.txt：
```CMakeLists.txt
cmake_minimum_required(VERSION 3.22.1)

project("tree-sitter-toml")

# @author android_zero
# Move up 4 levels from src/main/cpp (the current directory) to the tree-sitter-ndk directory
# Path deduction: cpp -> main -> src -> toml -> tree-sitter-ndk
get_filename_component(NDK_COMMON_DIR "${CMAKE_CURRENT_SOURCE_DIR}/../../../.." ABSOLUTE)

# Introduce the general configuration (set the TS_DIR and include paths)
include("${NDK_COMMON_DIR}/common-config.cmake")

# Make sure that the compiler can find the manually created "ts_toml.h" and "ts__onload.h" files.
include_directories(${CMAKE_CURRENT_SOURCE_DIR})

# Define source file
# parser.c: Tree-sitter The generated grammar parsing code
# scanner.c: Handwritten external scanner code
# tree-sitter-toml.cpp: JNI Binding code
# ${TS_DIR}/lib/src/lib.c: Tree-sitter Core runtime code
add_library(${CMAKE_PROJECT_NAME} SHARED
        "${TS_DIR}/lib/src/lib.c"
        parser.c
        scanner.c
        tree-sitter-toml.cpp
)

#  Link the Android log library
if (${CMAKE_SYSTEM_NAME} STREQUAL Android)
    find_library(log-lib log)
    target_link_libraries(${CMAKE_PROJECT_NAME} ${log-lib})
endif()
```
