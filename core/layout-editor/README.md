使用布局编辑器，通过将界面元素拖到可视化布局编辑器中，轻松构建 Android 应用布局。

### 特征
- [X] 拖放：布局编辑器允许开发人员轻松地将 UI 元素拖放到设计画布上，从而轻松创建应用程序布局的可视化表示。
- [X] 所见即所得的编辑：布局编辑器提供了所见即所得 （WYSIWYG） 界面，允许开发人员预览他们的应用程序在不同设备上的外观。
- [X] 蓝图模式：除了所见即所得模式外，布局编辑器还具有蓝图模式，允许开发人员使用布局的线框视图。
- [ ] 设计工具：布局编辑器提供了一系列设计工具，包括对齐参考线、对齐网格功能和一系列布局模板，以便更轻松地创建具有专业外观的布局。
- [ ] 主题编辑：开发人员可以通过从一系列预构建主题中进行选择或创建自己的自定义主题来修改其应用布局的主题。
- [X] 组件属性：布局编辑器提供了一系列开发人员可以修改的组件属性，包括大小、填充、边距等。
- [X] 预览模式：开发人员可以在智能手机、平板电脑和可穿戴设备等一系列设备上预览其应用程序的布局，以确保它在所有屏幕尺寸上看起来都不错。
- [ ] 代码编辑：布局编辑器还允许开发人员切换到代码模式并手动编辑布局的 XML 代码。
- [ ] 无障碍工具：布局编辑器包含一系列无障碍工具，可帮助开发者确保所有用户（包括残障用户）都可以访问其应用的布局。
- [ ] 布局验证：布局编辑器包含一个布局验证工具，可以识别常见的布局问题并提出解决方法。
- [x] 可拖拽约束布局: 与Android studio 一样的约束布局效果
**已知问题**：
1. 只添加了几个小部件。更多将很快添加。
2. 并非所有小部件的属性都可用。
